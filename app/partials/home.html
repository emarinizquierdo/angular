¿Qué es Angular.js?

HTML es bueno para declarar documentos estáticos pero falla cuando nosotros intentamos usarlo para declarar vistas dinámicas en aplicaciones web. Gracias a Angular.js podemos extender el lenguaje HTML en nuestras aplicaciones. El resultado es un entorno extraordinariamente expresivo, legible y fácil de desarrollar.

Lo básico

Edítame 
<!doctype html>
<html ng-app>
  <head>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.0.5/angular.min.js"></script>
  </head>
  <body>
    <div>
      <label>Name:</label>
      <input type="text" ng-model="yourName" placeholder="Enter a name here">
      <hr>
      <h1>Hello {{yourName}}!</h1>
    </div>
  </body>
</html>

Hola {{yourName}}!

¿Qué es Bootstrap?

Bootstrap es un framework que simplifica el proceso de creación de diseños web combinando CSS y JavaScript. Ha sido desarrollado por Twitter que recientemente liberó su versión 2.0. La mayor ventaja es que podemos crear interfaces que se adapten a los distintos navegadores (responsive design) apoyándonos en un framework potente con numerosos componentes webs que nos ahorrarán mucho esfuerzo y tiempo.

¿Qué es Node.js?

Node.js es un entorno de programación en la capa del servidor basado en el lenguaje de programación Javascript, con I/O de datos en una arquitectura orientada a eventos, y basado en el motor Javascript V8. Fue creado con el enfoque de ser útil en la creación de programas de red altamente escalables, como por ejemplo, servidores web. Fue creado por Ryan Dahl en 2009, y su evolución está apadrinada por la empresa Joyent, que además tiene contratado a Dahl en plantilla.
